# Persona Images

Add persona images to this folder with the following names:

- `emma.jpg` - Marketing Manager persona
- `max.jpg` - Freelance Graphic Designer persona  
- `lisa.jpg` - Stay-At-Home Mom persona
- `zoe.jpg` - College Student persona
- `sofia.jpg` - Hair Stylist persona
- `nina.jpg` - Nurse persona

## Image Requirements
- Format: JPG or PNG
- Recommended size: 400x300px or similar aspect ratio
- File size: Keep under 500KB for optimal loading

## Usage
These images will be displayed in 120px height containers with rounded corners and hover effects on the demo page persona selection step.
